# What is Aweww?
 Awesome EWW extension base on EWW with wonderful features!

# Improve EWW / SHR with SHRFACE
 The [shrface] extends `shr` / `eww` with org features and analysis capability.

# Improve EWW / SHR with SHR-TAG-PRE-HIGHLIGHT
 The [shr-tag-pre-highlight] extends `shr` / `eww` with syntax highlighting code block in HTML.

# Installing with Quelpa
 If you prefer to use a package manager, you can use [quelpa-use-package].

 ```elisp
 ;; Install Aweww
 (use-package aweww
   :quelpa (aweww :fetcher github :repo "GabrielFrigo4/aweww"))
 ```

# Installing with Elpaca
If you prefer to use a package manager, you can use [elpaca-use-package].

```elisp
;; Install Aweww
(use-package aweww
  :ensure (:type git :host github :repo "GabrielFrigo4/aweww"))
```
